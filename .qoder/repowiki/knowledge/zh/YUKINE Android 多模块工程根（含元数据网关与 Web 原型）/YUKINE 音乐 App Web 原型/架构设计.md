单文件组件结构：src/App.jsx 内集中定义 StatusBar、Search、CoverStack、HomeScreen、LibraryScreen、Placeholder、NowBar、BottomNav、Sheet 等子组件，并通过 App 根组件以 useState 管理 active（home/library/playing/settings）、isPlaying、query、toast、sheet 五类状态，作为唯一数据源向下分发；src/main.jsx 通过 createRoot 挂载到 index.html 的 #root。资源按用途分置于 public/assets/（专辑封面）与 public/fonts/（NotoSansCJKSC-Regular.otf、Outfit.ttf），由 A = '/assets/' 常量统一前缀引用。Vite 配置在 vite.config.mjs 中启用 @vitejs/plugin-react、预优化 react/react-dom/client，并将开发服务器 host 绑定 0.0.0.0、允许 terminal.local 访问，配合 warmup.clientFiles 加速冷启动。图标全部来自 @phosphor-icons/react，无路由或网络层，所有交互通过 notify 回调触发 toast 提示。
