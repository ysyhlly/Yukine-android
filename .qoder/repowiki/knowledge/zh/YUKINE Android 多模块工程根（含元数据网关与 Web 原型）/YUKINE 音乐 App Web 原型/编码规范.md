- 组件以函数式声明并就近导出，所有状态集中在 App 根组件，子组件仅接收 props 回调，不持有独立 state。
- 可复用 UI 片段（StatusBar、Search、CoverStack、NowBar、BottomNav、Sheet）从业务页面中抽离为独立函数组件，通过 props 注入行为。
- 静态列表数据（recent、libraryTiles、personalRows、tabs）以顶层常量数组形式定义，模板侧用 .map() 渲染，避免硬编码 JSX。
- 图标统一通过 @phosphor-icons/react 按需导入并以 Icon size weight 属性控制样式，不使用 CSS 自定义图标。
- 图片资源通过 const A = '/assets/' 常量拼接路径，避免在多处重复写死 /assets/ 前缀。
- 用户反馈采用统一的 notify(message) 模式：设置 toast 状态后通过 window.__yukineToast 定时器 1800ms 自动清除，替代 alert 或第三方通知库。
