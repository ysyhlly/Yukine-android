- DI 模块统一放在 `di/` 包内，使用 `@Module @InstallIn(SingletonComponent::class) object XxxModule` 形式，通过 `@Provides @Singleton` 暴露接口实现。
- UI 组件通过 `@AndroidEntryPoint` 注入，并由 `MainActivityComposition` 按 feature 拆分为 `features.xxx.initialize(...).bindRoot(...)` 的链式装配模式。
- 播放相关状态/命令采用 Owner 模式：大量 `Playback*Owner` 文件各自封装单一职责（队列、通知、歌词、可视化等），由 `PlaybackServiceRuntime` 组合。
- 测试按 sourceSet 分层：纯 Kotlin/Java 逻辑放 `test/`，需要 Android 框架的放 `androidTest/`，命名以 `*Test.kt` / `*InstrumentedTest.kt` 结尾。
