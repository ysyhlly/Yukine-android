- 所有对外请求统一经 `context.transport.getJson()` 发起，禁止在 core 中直接调用 fetch，以便在 Node 下替换为 SQLite 缓存层。
- 上游响应一律通过 `object()/array()/string()/number()` 等安全转换函数解析，拒绝强类型断言，保证对未知字段健壮。
- 结果通过 `result(body, status, trace, maxAge?)` 构造，统一设置 `Content-Type`、`Cache-Control` 与 `X-Content-Type-Options` 响应头。
- 每个端点内部维护 `AttemptSummary { attempted, reachable }`，当所有上游均不可达时返回统一的 502 upstream_failure。
