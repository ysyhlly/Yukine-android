# 今日聆听轨迹设计 QA

## 对照基准

- 修改前截图：`tmp/home-classic-today-chart.png`
- 最终实现截图：`tmp/final-phrases.png`
- 滚动聚焦截图：`tmp/final-phrases-focus.png`
- 全页对比：`tmp/home-listening-trail-comparison.png`
- 重点区域对比：`tmp/home-listening-trail-focus-comparison.png`
- 验证视口：1080 × 2400。
- 验证状态：经典布局、浅色主题、一首本地歌曲、今日已有播放记录；“最近播放”保留在轨迹下方。

## 视觉与文案核验

- 标题固定为“今日聆听轨迹”；辅助文案改为更自然的“今天听过的，都落进了旋律里”，并保留“留下一段回声”的艺术感，不展示精确时段统计或专业术语。
- 五线谱作为水平静态背景，不再随播放数据起伏；中间蓝色轨迹线已按反馈完全移除。
- 每个小时以该小时的聆听分钟数决定显示强度：0 分钟为下限，60 分钟为上限，超过 60 分钟仍按满幅显示；平方根映射让短时聆听保持可见。
- 相邻小时使用三次 B 样条生成平滑包络，降低单点尖峰和视觉方差；未来时段按 0 分钟处理。
- 三组正弦函数以 0.55 / 0.30 / 0.15 权重叠加，仅控制音符高亮节奏与密度，不再绘制成独立曲线，也不改变旋律音高。
- 音符采用《欢乐颂》中段高潮的相对音高序列，整体尺寸进一步放大；浅灰底谱始终保留完整旋律，蓝色数据层按时长提高亮度和密度。
- 乐句复杂度依次支持四分音符与休止符、横梁八分音符、附点与连音线、双音/三和弦、短琶音；末尾以高音 G 的空心二分音符和延音线收束。
- 时间标签改为“清晨 / 正午 / 黄昏 / 深夜”，保持叙事感；颜色、字体、卡片和间距沿用现有主题。
- “最近播放”没有移除，只下移到轨迹卡片之后，其数据和交互保持不变。

## 对比迭代记录

- 初版：使用精确时间刻度和硬直线，视觉更像统计图。
- 第二版：改为柔和曲线与艺术化时段文案。
- 第三版：加入随强弱和趋势变化的音符，但单次播放仍形成较窄尖峰。
- 第四版：扩大拟合邻域并收拢纵向范围，曲线起伏变得舒缓。
- 第五版：五条谱线整体随正弦函数起伏，并以《欢乐颂》开头旋律承载音符位置。
- 第六版：五线谱恢复为静态背景，数据轨迹改为三重正弦叠加；每小时 60 分钟封顶，音符换为《欢乐颂》高潮段。
- 最终版：取消中央蓝线并放大音符；B 样条包络和复合正弦改为控制音符亮度、密度与乐句复杂度，完整旋律轮廓保持不变。
- 数据修正：远端主页状态同样合并本地今日播放轨迹，避免周回顾有数据而今日轨迹为空。
