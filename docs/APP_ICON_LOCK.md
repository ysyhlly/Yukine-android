# ECHO App Icon Lock

The app launcher icon must stay visually identical to:

`app/src/main/res/drawable-nodpi/ic_echo_launcher_source.png`

Do not replace or redesign the launcher icon during UI, onboarding, cover fallback, notification, widget, or theme-icon work unless the user explicitly provides a new icon and asks to change it.

Generated launcher densities:

- `app/src/main/res/mipmap-mdpi/ic_echo_launcher.png`
- `app/src/main/res/mipmap-hdpi/ic_echo_launcher.png`
- `app/src/main/res/mipmap-xhdpi/ic_echo_launcher.png`
- `app/src/main/res/mipmap-xxhdpi/ic_echo_launcher.png`
- `app/src/main/res/mipmap-xxxhdpi/ic_echo_launcher.png`
