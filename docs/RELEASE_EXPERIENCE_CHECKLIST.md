## 2. 首次启动与授权

| 场景 | 步骤 | 通过标准 | 结果 |
| --- | --- | --- | --- |
| 全新安装启动 | 卸载旧包，安装候选 APK，首次启动。 | 应用不崩溃；无音乐/权限状态有标题、说明和明确按钮。 |  |
| 音频权限 | 点击授权入口并授予音频权限。 | 授权后可继续扫描或导入；不会停留在空白页。 |  |
| 通知权限 | Android 13+ 授予或拒绝通知权限。 | 两种选择都不影响前台播放；状态文案清楚。 |  |
| 无音乐设备 | 在无本地音乐设备启动。 | 显示无音乐状态和下一步操作，不出现技术错误。 |  |
| 冷启动恢复 | 结束进程后重新打开。 | 恢复上次设置、队列/播放位置按预期加载。 |  |

## 3. 曲库、导入与歌单

| 场景 | 步骤 | 通过标准 | 结果 |
| --- | --- | --- | --- |
| 本地扫描 | 授权后扫描 MediaStore。 | 曲库列表出现歌曲；专辑、艺术家、文件夹分组数量合理。 |  |
| 文件导入 | 通过系统文件选择器导入 1-3 个音频文件。 | 导入后可播放；重复导入不产生异常重复。 |  |
| 文件夹导入 | 通过系统目录选择器导入包含子目录的音乐文件夹。 | 子目录音频可被识别并持久化。 |  |
| M3U/M3U8 导入 | 导入本地歌单文件。 | 歌单条目解析成功；本地文件和流 URL 均可展示。 |  |
| 歌单操作 | 创建、重命名、删除歌单，向歌单添加/移除歌曲。 | 操作后列表即时刷新；重启后数据仍在。 |  |
| 歌单导出 | 导出当前歌单为 M3U8。 | 系统文件创建成功；导出文件可再次导入。 |  |
| 长列表体验 | 使用较大曲库滚动、搜索、切换分组。 | 滚动顺滑；搜索/排序/分组入口可发现；歌曲行不拥挤。 |  |

## 4. 播放器与队列

| 场景 | 步骤 | 通过标准 | 结果 |
| --- | --- | --- | --- |
| NowBar 展开 | 从底部 NowBar 打开完整播放页。 | 进入唯一完整播放器；主 Tab 不额外暴露播放页。 |  |
| 播放控制 | 执行播放/暂停、上一首、下一首、拖动进度。 | 控制即时生效；进度和时长显示正确。 |  |
| 队列入口 | 在播放页打开队列，调整或清空队列。 | 队列入口可达；空队列状态有明确文案。 |  |
| 更多操作 | 在播放页触发收藏、加入歌单、下载/更多等入口。 | 入口不遮挡主控制；失败时有清楚提示。 |  |
| 播放失败 | 播放无效 URI、缺失文件或断流 URL。 | 不崩溃；显示可理解的失败状态和恢复路径。 |  |
| 随机/重复 | 切换随机、关闭重复、全部重复、单曲重复。 | 图标/状态和实际播放行为一致，重启后设置按预期保留。 |  |
| 睡眠定时 | 设置 15 分钟或更短测试值，取消一次。 | 倒计时显示正确；取消后不再停止播放。 |  |

## 6. 流媒体与远程音源

| 场景 | 步骤 | 通过标准 | 结果 |
| --- | --- | --- | --- |
| 未登录状态 | 清空流媒体凭据后进入音源入口。 | 看到连接网易云/音源账号入口；不直接暴露 Cookie 等技术词。 |  |
| 登录/连接 | 配置可用账号或网关后刷新。 | 显示已连接状态；可进入我的歌单、每日推荐、心动推荐等主路径。 |  |
| 技术设置降噪 | 找到音质、网关、同步等设置。 | 技术项位于设置二级/高级路径，主界面不打扰普通用户。 |  |
| 流媒体播放 | 播放网易云或其他源歌曲。 | 能解析播放 URL；播放、切歌、进度、队列正常。 |  |
| 流媒体波形 | 播放未缓存完的流媒体歌曲。 | 早期有占位或部分波形；不等到歌曲过半才出现有效反馈。 |  |
| 网络断开 | 播放流媒体时断网再恢复。 | 失败提示可理解；恢复网络后可重试或自动恢复。 |  |

## 7. 后台、系统媒体与稳定性

详细逐项矩阵见 [PLAYBACK_SERVICE_STABILITY_MATRIX.md](PLAYBACK_SERVICE_STABILITY_MATRIX.md)。本节作为发布清单摘要，矩阵中的 P0 失败应阻断发布。

| 场景 | 步骤 | 通过标准 | 结果 |
| --- | --- | --- | --- |
| 后台播放 | 播放中按 Home、锁屏、再返回。 | 音乐持续；通知/锁屏控制可用；返回后 UI 状态同步。 |  |
| 通知控制 | 在通知中播放/暂停/上一首/下一首。 | 控制生效；通知文案和封面正确。 |  |
| 耳机断开 | 有线或蓝牙输出播放时断开。 | 按预期暂停或处理音频变更；不外放失控。 |  |
| 蓝牙切换 | 手机扬声器、蓝牙耳机/车机间切换。 | 播放不中断或能明确恢复；媒体控制仍可用。 |  |
| 媒体焦点 | 让其他音乐/视频 App 抢占音频焦点。 | Echo 按焦点规则暂停/降低音量/恢复。 |  |
| 来电/通话 | 播放中模拟来电或通话场景。 | 播放状态符合系统预期；通话结束后可恢复。 |  |
| 进程被杀 | 后台播放时杀进程或系统回收后重新进入。 | 不出现崩溃循环；可重新建立播放服务。 |  |
| 超大队列 | 加载大于 500 首歌曲的队列。 | 队列操作可用；切歌和滚动无明显卡顿。 |  |
